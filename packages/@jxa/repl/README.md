# @jxa/repl

REPL for JXA.

## Install

Install with [npm](https://www.npmjs.com/):

    npm install -g @jxa/rep

## Usage

    $ npx @jxa/repl
    # or
    $ npm install -g @jxa/repl
    $ jxa-repl
    > Application("System Events").currentUser().name();

## Command

### `.clear`

Clear current buffer and history.
It will release defined variables.
